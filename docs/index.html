Angular Modules Documentation

A project containing a few modules that aid in the use of common js libraries. To use the modules you might need to link in css files and js libraries they depend on.

Modules

CodeMirror (codemirror)

CodeMirror resides in the module codemirror and allows you to mark textarea elements to be syntax highlighted by CodeMirror. You can also, optionally, set the cm-options attribute to set options.

It's also possible to keep your model in sync with the CodeMirror instance by setting the cm-model attribute.

Type code in the textarea below to see how it automatically is reflected in the CodeMirror instance, and vice versa!

jQuery UI (jquery-ui)

The jQuery UI module contains directions used to transform elements based on the jQuery UI functionality.

All jQuery UI directives are prefixed by jui and reside in the jquery-ui module.

Button

By using the jui-button directive it's possible to transform links and buttons into jQuery UI buttons. All you need to do is to add the jui-button attribute to the target element.
